# CodeForge - Python dependencies
#
# The editor core is 100% Python standard library (tkinter/ttk, ast,
# importlib, subprocess, ...) and needs nothing from this file just to
# run `python main.py`. Everything below is an OPTIONAL feature that
# CodeForge detects at runtime via try/except and gracefully degrades
# without - install only the ones you actually want:
#
#   tkinterdnd2   -> drag-and-drop file opening onto the tab bar /
#                    Explorer panel. Without it, drag-and-drop is
#                    silently disabled; everything else works as normal.
#
#   yt-dlp
#   python-vlc    -> the background Music tab (stream + play audio from
#                    a YouTube URL). Without both installed, the Music
#                    tab shows an install hint instead of transport
#                    controls. python-vlc is just Python bindings - it
#                    also needs the real VLC application installed
#                    separately: https://www.videolan.org/vlc/
#
# Install everything with:
#     pip install -r requirements.txt

tkinterdnd2>=0.3.0
yt-dlp>=2024.1.0
python-vlc>=3.0.18000

# ---------------------------------------------------------------------
# Not pip-installable, but worth knowing about - CodeForge shells out to
# these system tools for a couple of other optional features and just
# skips the feature if they're missing from PATH:
#
#   git   -> the Source Control tab (staged/unstaged file list, diff
#            preview, commit/push/pull, branch switching). Get it from
#            https://git-scm.com/ if `git` isn't already on PATH.
#
#   node  -> real syntax checking for JavaScript files. Without it,
#            JS falls back to a generic bracket/quote-balance scanner
#            instead of node --check.
